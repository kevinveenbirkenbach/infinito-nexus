# Sovereign Local AI Platform

## Purpose
A privacy-first foundation for enterprises running AI locally and keeping sensitive data on-prem.

## Target Audience
Enterprises, research teams, and regulated organizations that need AI capabilities without sending data to external clouds.

## Included Components
- web-app-minio
- web-svc-libretranslate
- web-app-keycloak
- svc-prx-openresty

## Architecture Overview
The stack provides on-prem data storage, a local AI-adjacent service, centralized identity,
and secure exposure through a reverse proxy. This bundle is a skeleton activation map.
