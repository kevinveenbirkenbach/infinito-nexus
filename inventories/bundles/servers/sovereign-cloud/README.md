# Sovereign Cloud Platform

## Purpose

A fully sovereign private cloud infrastructure designed for organizations that
require full control over their data, identity, and service stack.

## Target Audience

Public institutions, regulated industries, European organizations,
and enterprises seeking a compliant cloud alternative.

## Marketing Positioning

- Sovereign Cloud
- Data Ownership
- Compliance-Ready Infrastructure
- Zero Vendor Lock-In
- Self-Hosted Cloud Platform

## Included Components

- web-app-nextcloud
- web-svc-onlyoffice
- web-app-keycloak
- web-app-minio
- web-app-matrix
- svc-prx-openresty

## Architecture Overview

All services run behind a reverse proxy and share centralized identity management.
The inventory defines a single host and activates roles declaratively.
