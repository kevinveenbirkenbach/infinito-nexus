# Sovereign Cyber Defense Stack

## Purpose
A cyber-resilient sovereign infrastructure foundation for high-assurance environments.

## Target Audience
Defense industry, government agencies, critical infrastructure operators, and regulated environments.

## Included Components
- web-app-keycloak
- web-app-matrix
- svc-prx-openresty

## Architecture Overview
Central identity and secure communications are deployed behind a hardened reverse proxy.
This bundle is a skeleton activation map and contains no application-specific configuration.
