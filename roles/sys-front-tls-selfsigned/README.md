# sys-front-tls-selfsigned

Self-signed TLS provider for sys-front-tls (SAN aware).

## Inputs

- tls_domain
- application_id
- tls_selfsigned_base
- tls_selfsigned_days
- tls_selfsigned_key_bits
- tls_selfsigned_subject (C/O/OU/CN)
