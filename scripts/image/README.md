# Image Scripts

This directory contains scripts related to the image lifecycle used by the project.

- 🏗️ Build container images for local and CI workflows
- 📦 Publish, retag, and maintain image artifacts in registries
- 🔧 Keep image-focused automation grouped in one place

The scope of this folder is image handling.
Workflow orchestration, metadata resolution, and generic waiting logic should live in other script directories.
