# Wait Scripts

This directory contains generic wait and polling scripts used by workflow automation.

- ⏳ Wait for asynchronous resources to become available
- 🔎 Poll external systems until expected artifacts are ready
- 📦 Wait for CI and mirror images to become available in registries
- 🧭 Centralize readiness checks outside build and release scripts

The scope of this folder is waiting and availability checks.
Build, release, and resource-specific lifecycle operations should stay in their dedicated directories.
