# Tests 🧪

All automated tests for Infinito.Nexus live here.

| Directory | Type | Guide |
|---|---|---|
| `unit/` | Unit tests (isolated, no I/O) | [unit.md](../docs/contributing/testing/unit.md) |
| `integration/` | Integration tests (real component boundaries) | [integration.md](../docs/contributing/testing/integration.md) |
| `e2e/` | End-to-end Playwright tests | [playwright.md](../docs/contributing/testing/playwright.md) |
| `regression/` | Regression guard rules | [regression.md](../docs/contributing/testing/regression.md) |
| `lint/` | Structural lint checks | [lint.md](../docs/contributing/testing/lint.md) |

Run each suite via the `make` targets documented in the linked guides.
