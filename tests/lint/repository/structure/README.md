# Structure Lint 📏

Repository-wide structural limits: per-folder item count, per-file Python line cap, and inline shell-script block size.

Tests in this directory MUST only cover size/count thresholds on tracked files and folders. Naming, content, and dependency rules MUST live under their respective sibling clusters.

For framework and `make test-lint` usage see [lint.md](../../../../docs/contributing/actions/testing/lint.md).
