# Shell Lint 🐚

Shell-script conventions: ban on inline multi-line PHP blocks embedded in `.sh` files.

Tests in this directory MUST only cover `.sh` source patterns. Python rules MUST live under [`python/`](../python/); YAML/Jinja rules MUST live under [`yaml/`](../../repository/yaml/).

For framework and `make test-lint` usage see [lint.md](../../../../docs/contributing/actions/testing/lint.md).
